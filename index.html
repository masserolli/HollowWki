Focando na gameplay de Hollow Knight, o jogo é em 2D, com controle básico de movimento usando as teclas "A" e "D" (ou os direcionais do controle). Além disso, o botão de pulo é a barra de espaço, e "W" serve para mirar ataques ou magias para cima, enquanto "S" ou o direcional para baixo é usado para ataques para baixo.
